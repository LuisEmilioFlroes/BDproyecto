### 3. `BDproyecto` 
Contiene las entidades de base de datos y repositorios JPA. Es utilizado por los otros dos módulos.

- **Tecnologías:**
  - Spring Data JPA
  - Hibernate Validator
  - Lombok
  - Jackson
  - Jakarta Validation

- **Entidades:**
  Tareas con jerarquía y subtareas.
  Usuarios con correo y contraseña.
   Historial de acciones por tarea.
